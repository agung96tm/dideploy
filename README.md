Dokumentasi K3s untuk deploy Kubernetes di server (bare metal / VPS). Publik dan terbuka untuk kontribusi.

## Jalankan lokal

```bash
pip install -r requirements.txt
mkdocs serve
```

Buka http://127.0.0.1:8000 di browser.

## Kontribusi

Siapa saja boleh berkontribusi. Cara singkat:

1. Fork repositori ini
2. Ubah konten di `docs/` (tambah entri di `mkdocs.yml` jika menambah halaman baru)
3. Buat pull request
